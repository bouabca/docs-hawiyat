The Hawiyat CLI can be used to create, deploy, and manage projects.

## Requirements

Is required to be already authenticated with the Hawiyat CLI.

## Commands

1. `Hawiyat project create` - Create a new project.
2. `Hawiyat project info` - Get information about a project.
3. `Hawiyat project list` - List all projects.
