The Hawiyat CLI can be used to create, deploy, and manage applications.

## Requirements

Is required to be already authenticated with the Hawiyat CLI.


## Commands

1. `Hawiyat app create` - Create a new application.
2. `Hawiyat app delete` - Delete an application.
3. `Hawiyat app deploy` - Deploy an application.
4. `Hawiyat app stop` - Stop a running application.
