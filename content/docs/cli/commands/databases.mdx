title: Databases
description: A guide to using the Hawiyat CLI to manage databases
---

The Hawiyat CLI can be used to create, deploy, and manage databases.

## Requirements

Is required to be already authenticated with the Hawiyat CLI.


## Commands

### MariaDB 

1. `Hawiyat database mariadb create` - Create a new mariadb database.
2. `Hawiyat database mariadb delete` - Delete an mariadb database.
3. `Hawiyat database mariadb deploy` - Deploy a mariadb database.
4. `Hawiyat database mariadb stop` - Stop a running mariadb database.
