---

Hawiyat CLI is a command-line tool for remotely managing your Hawiyat server. It simplifies creating, deploying, and managing applications and databases.

## Installation

```bash
npm install -g @Hawiyat/cli
```

## Usage

```bash
Hawiyat COMMAND
```
To get help on a specific command:

```bash
Hawiyat COMMAND --help
```
